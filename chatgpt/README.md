## 取り組み方のヒント

* **順番厳守より「理解度基準」**：飛ばしても戻っても OK。
* **小さく動かしテストを書く**：再帰はバグが潜みやすい。
* **メモ化／DP への写像**：同じ課題でも「キャッシュあり」「なし」で速度差を観察。
* **再帰深度の限界**：Python なら `sys.setrecursionlimit` を試す。
* **ビジュアル化**：トレース木やフラクタル画像を描くと動きが見える。
